Next, we create the **IPizzaBuilder** interface. 

```
IPizzaBuilder
```

This interface defines all the steps needed to create a Pizza.

```
void BuildDough();
void BuildSauce();
void BuildTopping();
Pizza GetPizza();
```
here we require **BuildDough**, **BuildSauce**, **BuildTopping** and **GetPizza** to make a **Pizza**
